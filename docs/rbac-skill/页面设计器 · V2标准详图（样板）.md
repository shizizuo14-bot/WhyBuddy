# 页面设计器 · V2 标准详图（样板）

> 本系统在 V2 中为**执行点（PEP）**。保留它真正独有的：画布/组件/Schema/渲染流水线/预览。
> V2 相对 V1 的关键改动：
> - **P0-1**：`PERMISSION_RENDER` 的**判定**委托 PDP（页面/区块/组件/字段显示控制由 PDP 决策）。
> - **P0-3**：`BINDING_SCHEMA` 的字段改为**绑定数据中台 SSOT 字段**，不再自定义。
> - **P0-2**：本地 `DEPENDENCY_GRAPH/INVALIDATION` 并入**全局依赖图**；发布事件进总线。
> - **P1-7**：本地 `POLICY_LEDGER` 改为写**统一 Trace**。
