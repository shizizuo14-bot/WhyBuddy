# 数据中台 · V2 标准详图（样板）

> 本系统在 V2 中**就是内核 ② 字段 SSOT 的本体**——实体/字段/关系的唯一事实源，对外供绑定。
> V2 相对 V1 的关键改动：
> - **拆分（P2-8）**：OLTP「数据建模器 = SSOT」留在中台；OLAP 数仓（RAW→ODS→DWD→DWS→ADS + 联邦/语义/物化）**拆为独立产品**，本图不再混入。
> - **权限不在这里决策（P0-1）**：`MODEL/ROW/FIELD_PERMISSION` 的**定义**保留（属于策略），但**判定委托 PDP**。
> - 模型变更进事件总线、被全局依赖图失效；数据访问决策证据进统一 Trace。
