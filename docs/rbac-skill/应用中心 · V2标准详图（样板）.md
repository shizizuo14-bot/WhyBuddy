# 应用中心 · V2 标准详图（样板）

> 本系统在 V2 中**就是内核 ⑥ 组装根的本体**——把页面/工作流/数据/角色/AIGC 捆成一个可发布应用。
> V2 相对 V1 的关键改动：
> - **P1-5 坐实**：把空泛的 `PERMISSION/DEPENDENCY_VALIDATE` 落成「**跨系统引用闭包校验**」——`app.workflow.assignee ∈ app.roles`、`app.page.binding ∈ app.dataModels`，调**全局依赖图**。
> - **P1-6 明确**：`Release Artifact` **钉死各子产物版本**；运行时一律按快照解析，不取最新。
> - **P0-1/0-3/1-7**：鉴权委托 PDP；引用数据中台 SSOT；发布/失效进总线与依赖图；决策证据进统一 Trace。
> - 这一版可执行：`PUBLISH_GATE` 的闭包校验 = 代码里的 `orchestrator.publishGate()`。
